KEIBA_APP

競馬の出馬表と過去成績（近5走）を表示し、印（◎〇▲△×消）をつけて予想管理できる Streamlit アプリです。

☆主な機能
〇出馬表
枠番色の再現
馬名、性齢、騎手、枠番、馬番のカード表示
印（◎〇▲△×消）をセレクトで付けられる
付けた印は自動保存され、再起動しても復元

〇近走表示
各馬の直近レースをカード形式で横スクロール表示
着順ごとにカード色が変化  
  - 1着：金  
  - 2着：青  
  - 3着：赤
近5走は「消」印をつけると非表示になる
印によるフィルタリング  
人気順 / オッズ順 の並び替え

☆使い方
streamlit run app3.py
